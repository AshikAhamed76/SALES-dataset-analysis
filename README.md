# Abstract

Business Intelligence tools play a vital role in transforming raw business data into meaningful insights. This project focuses on the development of an interactive **Sales Dashboard using Power BI**. The dashboard provides a comprehensive analysis of sales performance across different countries, regions, and sales channels.

The dashboard enables users to monitor key performance indicators such as Total Revenue, Total Cost, Total Profit, and Units Sold. Various visualizations including KPI cards, bar charts, waterfall charts, line charts, and slicers are used to present business information effectively. The project helps organizations make data-driven decisions by identifying profitable markets, analyzing sales trends, and evaluating regional performance.

# Chapter 2: Literature Review

Business Intelligence (BI) refers to technologies and practices used for collecting, integrating, analyzing, and presenting business information.

Power BI is a business analytics tool developed by Microsoft that enables users to:

* Connect to multiple data sources.
* Transform and clean data.
* Create data models.
* Build interactive dashboards.
* Generate business reports.

Many organizations use Power BI for sales analysis because it provides real-time insights and advanced visualization capabilities.

# Chapter 5: System Design

## Data Flow

Sales Dataset → Data Cleaning → Data Modeling → Power BI Dashboard → Business Insights

## Dashboard Components

### KPI Cards

* Average Total Cost
* Maximum Units Sold

### Gauge Chart

Displays Total Revenue performance against the target.

### Bar Chart

Compares Total Revenue and Total Cost.

### Waterfall Chart

Shows Unit Price contribution by Region and Sales Channel.

### Funnel Chart

Displays Revenue, Cost, and Profit comparison.

### Line Chart

Shows Revenue generated by Country.

### Slicer

Country-based filtering for interactive analysis.
